# AGENTS

## Purpose
This repository is a small static web project with a single-page HTML app and supporting JavaScript and CSS.

## What to know
- No build tools, package manager, or framework are present.
- Files to edit:
  - `index.html` — main HTML structure
  - `script.js` — interactive logic and application behavior
  - `styles.css` — custom styling, currently minimal and based on Pico styling
- Use plain vanilla HTML/CSS/JavaScript unless the user explicitly asks for a framework.
- Keep changes lightweight and avoid adding unnecessary dependencies.

## Common tasks
- Fix or extend interactive behavior in `script.js`
- Update page structure or content in `index.html`
- Adjust layout and styles in `styles.css`

## How to proceed
1. Inspect the current code in `index.html`, `script.js`, and `styles.css`.
2. Make minimal fixes or enhancements needed for functionality, accessibility, and clarity.
3. Avoid inventing a build process or package configuration unless the user explicitly requests it.

## Notes for agents
- There is no existing README or documentation to link; use this file as the primary workspace guide.
- If asked to add tests or tooling, ask the user for the preferred setup.
